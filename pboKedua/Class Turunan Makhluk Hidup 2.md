## Features

- Constructor
- This
- Super

## Tech

- [NetBeans IDE 15](https://netbeans.apache.org/front/main/download/nb15/) - Aplikasi IDE atau Integrated Development Environment yang basis bahasanya itu Java dan berjalan di atas Swing.


## Requirements
- Java (18.0.2.1)
- NetBeans IDE 15
